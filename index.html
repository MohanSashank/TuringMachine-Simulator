About Turing Machines

A Turing Machine consists of:

  • Q — finite set of states
  • Σ — input alphabet
  • Γ — tape alphabet (Σ ∪ {⊔})
  • δ — transition function
  • q₀ — initial state
  • q_acc — accept state
  • q_rej — reject state

At each step, the machine reads the current symbol, writes a new symbol, moves the head left or right, and transitions to a new state.
